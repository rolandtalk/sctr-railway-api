# Standalone repo (Option B)

This folder **is** the standalone SCTR Railway API. It has nothing to do with any parent project (e.g. Stockshopping).

- **GitHub:** [rolandtalk/sctr-railway-api](https://github.com/rolandtalk/sctr-railway-api)
- **Railway:** New Project → Deploy from GitHub → **rolandtalk/sctr-railway-api** → **Root Directory = empty** → Generate Domain

You can move this folder anywhere (e.g. `~/projects/sctr-railway-api`) and keep working:

```bash
cd /path/to/sctr-railway-standalone   # or wherever you moved it
git add -A && git commit -m "your message" && git push origin main
```
